# Network-Monitoring-Tool
Ferramenta de automação para monitoramento de conectividade de servidores e análise de disponibilidade em ambientes de TI
# 🛠️ IT Operations Toolbox

Este repositório reúne ferramentas e scripts desenvolvidos para otimizar processos de **Operações de TI**, monitoramento de infraestrutura e suporte técnico N2. O foco é a automação de tarefas rotineiras e a agilidade no diagnóstico de falhas (troubleshooting).

## 🚀 O que este projeto resolve?

Em ambientes críticos (como o setor financeiro), a disponibilidade de serviços é prioridade. Estas ferramentas ajudam a:
* Reduzir o tempo de resposta em caso de quedas de servidores.
* Identificar padrões de erro em grandes volumes de dados.
* Automatizar a geração de logs para auditoria e conformidade.

---

## 📂 Estrutura do Repositório

### 1. 🐍 Monitoramento de Conectividade (Python)
* **Arquivo:** `check_servers.py`
* **Função:** Verifica a disponibilidade de múltiplos IPs/Hosts simultaneamente.
* **Destaque:** Compatível com ambientes híbridos (**Windows e Linux**) e gera relatórios de status automaticamente em arquivos `.txt`.

### 2. 📊 Monitoramento de Incidentes (SQL)
* **Arquivo:** `monitoramento_transacoes.sql`
* **Função:** Consultas estruturadas para identificar falhas críticas, timeouts e gargalos de performance em bancos de dados de alta volumetria.
* **Destaque:** Foco em análise de causa raiz para suporte a "Salas de Guerra" e incidentes de performance.
